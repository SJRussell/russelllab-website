# Russell Research Lab Website

Static website for the Russell Research Lab at Wilfrid Laurier University.

Live site: https://russelllab.ca

## Structure

- `index.html`: homepage
- `research.html`: lab research themes
- `projects.html`: current project hub
- `publications.html`: selected publications
- `team.html`: current team and alumni
- `contact.html`: contact and future trainee inquiries
- `css/main.css`: shared styling
- `images/`: site images, headshots, project figures, logo, and favicon

## Deployment

The site is deployed with GitHub Pages from the `main` branch and uses the custom domain in `CNAME`.

## Content Notes

- Student emails are intentionally not published. Student and trainee inquiries should route through the Qualtrics contact form.
- Stewart's email is displayed on the contact page with light client-side assembly to reduce scraping.
- The project page is written as a grant-reviewer and collaborator-facing hub. Individual trainee projects should be kept in lab voice rather than first-person student voice.
